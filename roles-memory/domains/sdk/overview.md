# SDK / 工具链领域

## 定位

SDK 和 CLI 工具开发，涵盖 API 设计、包管理、VS Code 扩展、命令行工具。

## 核心概念

- **CLI 设计**: 命令结构、参数解析、输出格式
- **API 设计**: REST/gRPC/模块化接口
- **MCP (Model Context Protocol)**: Agent 工具协议
- **扩展开发**: VS Code Extension API
- **包管理**: npm, pip, 依赖管理

## 连接

- ← soc: 固件编译结果集成
- → debug: 诊断和调试接口
- → vendor: 厂商工具集成
- → all: 为所有领域提供工具支撑
