# 调试 / 诊断领域

## 定位

调试、诊断、日志、性能分析领域，涵盖从硬件到软件的全链路调试。

## 核心概念

- **硬件调试**: JTAG, SignalTap, ILA, 逻辑分析仪
- **软件调试**: GDB, 日志, 断言, 性能分析
- **诊断流程**: 定位→隔离→分析→修复→验证
- **日志规范**: 级别、格式、上下文信息
- **性能分析**: profiling, 瓶颈识别, 火焰图

## 连接

- ← fpga: 上板调试
- ← soc: JTAG/OpenOCD 调试
- → sdk: 诊断工具和接口改进
